== Description ==

Add AudioSnaps to the available oEmbed providers.

Add any picture to your blog post or page just by adding the link. Example: http://audiosnaps.com/k/dhzr/

== Installation ==

1. Upload `AudioSnaps-oembed` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= What is AudioSnaps? =

AudioSnaps are pictures with sound, everything embedded in a single jpeg file
